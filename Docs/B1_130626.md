# Bitacora de desarrollo - APPS
## Día 1 (13 de julio de 2026)
### Objetivo del día
- Configurar el entorno de desarrolllo y probar la conexión con el ESP32.

### Metodología
Se utilizó un ESP32, un Display LCD, dos resistencias de 10 kΩ y dos potenciometros de perilla. Por medio de cables Dupont, se hicieron las debidas conexiones entre los dispositivos. Dichas conexiones se encuentran en las siguientes tablas:
|LCD|ESP32|
|---|-----|
|VCC|VIN  |
|GND|GND  |
|SDA|D21  |
|SCL|D22  |

|S1 |ESP32|
|---|-----|
|Pata izquierda|GND|
|Pata centro|D32|
|Pata derechas|3V3|

|S2 |ESP32|
|---|-----|
|Pata izquierda|GND|
|Pata centro|D33|
|Pata derechas|3V3|

El código (se encuentra en el repositorio) funcionó de manera correcta. Al girar las perillas se muestra en el Display el cambio de porcentajes para ambos potenciometros, lo cuál nos indica que tanto las conexiones como el código funcionan de manera correcta.

### Logros
- Se instaló el soporte para ESP32 en Arduino IDE.
- Se conectó de manera correcta el ESP32, el Display LCD, los potenciometros y las resistencias.
- El código funcionó de manera correcta junto con los componentes.

### Problemas encontrados
Si bien el código funcionó de manera correcta, hubo un error el cual dice que cuando ambos sensores se encuentran en un rango de ±10%, el motor se apaga, lo cual es contradictorio con el proposito de ambos sensores.
